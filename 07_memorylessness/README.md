# Memorylessness of the Exponential Distribution

Three figures:
1. **Density overlay:** f(t) and conditional densities f(t-s|T>s) for s=1,2,3.
   All identical exponentials, shifted.
2. **Timeline:** Poisson process with "now" line, already-waited/remaining-wait
   arrows staggered vertically.
3. **Histogram:** 200k samples, blue (all) vs orange (residuals T-2|T>2)
   overlap perfectly.

**Run:** `python memoryless.py`
