# Entrainment via Contraction Theory

2-species negative feedback loop with periodic forcing (Russo, di Bernardo,
Sontag, PLoS Comp Biol 2010). μ₂(J) ∈ [-1.65, -1.13] < 0 (contracting).
12 trajectories from diverse ICs all converge to the same periodic orbit.

Two panels: time traces + phase portrait.

**Run:** `python entrainment_contraction.py`
