# Logarithmic Norms

Three figures:
1. **Bounds:** ||exp(At)||_p vs exp(μ_p t) and β·exp(αt) for 1-norm and
   2-norm. A has positive log-norms despite being stable.
2. **Regions:** Stability region vs μ₂<0 region in parameter space.
3. **Contractive:** A with μ₂<0. Both bounds decay, shaded conservatism gap.

**Run:** `python log_norm.py` and `python log_norm_contractive.py`
