# Deficiency Theory (10 figures)

Three reaction network examples illustrating deficiency-zero and
deficiency-one theory:

- **Example 1 (δ=0):** Triangle A ⇌ B ⇌ A+B ⇌ A. Complex-balanced.
  Stationary distribution is product-of-Poissons. 4 figures.
- **Example 2 (δ=1):** ∅ ⇌ A+B ⇌ 2A, A ⇌ B. NOT complex-balanced.
  Checkerboard parity pattern. 4 figures.
- **Example 3 (Edelstein, δ=1):** Bistable system. 2 figures.

Complex-balance check figures show arrow thickness ∝ reaction rate,
with flow badges at each node.

**Run:** `python deficiency_theory_figures.py` (generates all 10 SVGs)
