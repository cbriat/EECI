# AIC vs AIRC (Rein Controller)

Comparison under a translation rate disturbance (γ: 1→2 at t=50).
- **AIC (k_d=0):** enters sustained oscillations (Hopf bifurcation).
- **AIRC (k_d=50):** rein stabilizes, converges to p*=1 within ~20 t.u.

Each figure has top panel (protein) and bottom panel (controller species).

**Run:** `python aic_vs_airc.py`
